Bands I've Seen

Bands I've Seen is a small application for cataloguing which bands you've seen live at gigs or festivals. It was written entirely in Java (the GUI was built using the Swing Framework), with all data stored on a sqlite database.

Features

Record what gigs or festivals you've been to with this convenient application. No installation required, just download the right file type for your Operating System and get to work! Details which can be associated with a gig or festival include:

  • Who you saw
  • How good the performance was
  • The date and location of the event
  • Who you went with

All saved gigs and festivals are then presented in a simple table, allowing you to easily see what has been added to the database. In addition to this, it's possble to filter the presented data on citeria such as the date, the location, who you went with, and whether the event was a gig or a festival. Results can be sorted by artist name, event date, location, or rating in ascending or descending order. It's never been easier to keep track of who you've seen.

Made a mistake with the details? No problem, all records can be edited from within the application. Change the rating of a performance, add more days to a festival, rename a friend, or whatever else you want. Acidentally added a gig twice whilst suffering from a hangover? It's possible to delete saved events as well. Don't worry, you'll be asked that you're sure you want to proceed to prevent any mishaps.

How to run

Either download the latest release from here, or clone the repository from here and build the jar yourself. Save the file to a directory and double click to run. Running the application for the first time will create a .db file in the same directory. Keep this with the application!
